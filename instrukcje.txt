Instrukcja obsługi treningu modelu do rozpoznawania dań

Wymagania:
Python wersja 3.10
requirements.txt

Struktura katalogów

Trenowanie/
├─model/
    ├─best_model.h5
    ├─converter.py
    ├─history_finetune.json
    ├─training_results.png
    ├─converted/
        ├─converted_model.tflite
├─dataset/
├─GPUtrening.py
├─instrukcje.txt
├─requirements.txt
├─sortowanie.py

1. Pobranie datasetu ze strony:
https://www.kaggle.com/datasets/dansbecker/food-101

Wypakowanie datasetu w folderze Trenowanie

2. Stwórz środowisko wirtualne:

W  PyCharm: Settings → Project → Python Interpreter → Add… → New Virtualenv (wskaż folder .venv), następnie „Install from Requirements” i wskaż plik requirements.txt z katalogu projektu.

Windows PowerShell:
py -3.10 -m venv .venv; ..venv\Scripts\Activate.ps1.

Windows cmd:
py -3.10 -m venv .venv && ..venv\Scripts\activate.

Linux/macOS (bash/zsh):
python3.10 -m venv .venv && source .venv/bin/activate.

3. Instalacja zależności

pip install -r requirements.txt.

4. Podział datasetu na train i test

python sortowanie.py

Program dzieli dataset na zbiór treningowy 80% oraz testowy 20% zgodnie z zaleceniami twórców otrzymanymi wraz z pobranym datasetem

5. Uruchomienie treningu

python GPUtrening.py

po skończeniu trenowania model będzie się znajdować w folderze model

6. Testowanie modelu

python model/test.py

prosty program z możliwością wybrania obrazu, wyświetla wyniki działania modelu.
